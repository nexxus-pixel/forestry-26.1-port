Forestry Translations
======================

Translations Guide for Forestry: Community Edition

### Encoding

Files must be encoded in UTF-8. Anything else will not work. Pull requests, submissions or contributions not encoded in UTF-8 will have to be rejected or reverted.

### Testing your language files

Put the files into your .minecraft directory into a subfolder "assets/" that mirrors that of the Forestry jar. Any correctly named localization file should take precedence over the defaults in the jar.

Alternatively you can also add your localization files to the Forestry .jar directly. 

### Notes

**IMPORTANT**: Forestry's English language is split into two files, one hand-written, and one autogenerated:
- `src/main/resources/assets/forestry/lang/en_us.json`
- `src/generated/resources/assets/forestry/lang/en_us.json`

To easily check if your language is missing translation entries, or has extras that aren't used anymore,
use the Python script included in this folder.

Run this command in a PowerShell or Command prompt (or your terminal of choice):
```sh
python check_translations.py ru_ru.json
```
